Arduino xPL S0 Energy Meter
=

Arduino based electric energy meter with ethernet (W5100) and xPL.

Uses Extra Libs
-

ooPinChangeInt

Features
-

* 6 independent S0 counters
* Divider per full unit per counter
* Persistent config in EEProm
* Ugly xPL support for
	* basic schema 
	* trigger
	* heartbeat
